Tug Boat Model
====
## Description
The tug boat model for Gazebo  

## Install  
```bash
$ sudo mkdir ~/.gazebo/models  
$ cd  
$ git clone  https://github.com/KoutaOhishi/Tug_boat_model.git  
$ sudo cp -r ~/Tug_boat_model ~/.gazebo/models/  
```

## Usage
 You can import tug boat model in Gazebo.
